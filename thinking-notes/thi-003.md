# ストーリーテリングの演繹法

- 前提
  - 演繹法
    - 鳥は飛ぶ + ハトは鳥 = ハトは飛ぶ
    - 一般解 + 一般解に含まれる個別の要素 = 個別の要素は一般解に従う
    - 命題Aが真 + 命題Bが真 = 命題Cは真
    - 前提が真 => 結論は真
    - 一般解 + 主人公が一般解に含まれることが真 = 主人公は一般解に従う
    - 主人公への期待 => 期待の実演
  - 帰納法
    - ハトは飛ぶ + スズメは飛ぶ = 鳥は飛ぶっぽい
- 弁証法では葛藤が話を面白くしていた
  - 弁証法
    - ハトは飛ぶ + ハトは飛ばないときもある = ハトは飛ぶが飛ばないときもある
  - 物語での弁証法
    - 主人公のこれまでの方法による成功 + 主人公のこれまでの方法が通用しないことによる失敗 = 主人公の失敗に学んだ新しい方法による成功
