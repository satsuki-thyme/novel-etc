## しかし重要なのは「背景→結果」は一対一ではない

同じ背景でも全く違う人物になります。

つまり

> 背景 × 本人の資質 × 出会い × 選択

* **赦し**
* **復讐**
* **使命を与えられる**

この「転換点」は、その後の行動原理に直結します。

---

## より本質的には「世界観」を形作る背景に分けられる

背景は、キャラクターが「世界とはどういう場所か」をどう信じるようになったか、という観点でも整理できます。

* **世界は優しい**（愛情・救済・信頼）
* **世界は残酷だ**（虐待・戦争・喪失）
* **世界は公平だ**（努力が報われた）
* **世界は不公平だ**（差別・理不尽）
* **世界は力が全てだ**（暴力・権力）
* **世界は絆で成り立つ**（友情・家族）
* **世界に意味はない**（虚無・絶望）
* **世界は変えられる**（革命・成功体験）

この「世界観」が、そのキャラクターの善悪や判断基準、葛藤の土台になります。そのため、単に「善い背景」「悪い背景」と分けるよりも、「どのような世界観を育んだ経験か」という視点で設計すると、より立体的で一貫したキャラクターを作りやすくなります。

> 「ルール通りか？」

---

## 実際の人は複数持っている

例えば

```
①家族
②正義
③名誉
④利益
```

という優先順位を持っていたら

家族を守るためなら

正義も捨てるし

利益も捨てます。

---

## さらに細かくすると「禁止」と「義務」がある

例えば

```
絶対に嘘はつかない
```

これは

禁止。

一方

```
困っている人は助ける
```

これは

義務。

この二つは性質が違います。

---

## 私なら「if→thenルール」にする

人は意外と

```
裏切られたら切る

恩を受けたら返す

侮辱されたら怒る

負けそうなら逃げる

子供なら助ける

困っていたら放っておけない
```

みたいな

**条件反射**

の集合です。

---

## そして、一番面白いのは「ルール同士の衝突」

例えば、

* 「嘘はつかない」
* 「仲間は守る」

という二つのルールを持つ人物が、

敵に「仲間はどこだ？」と聞かれたらどうするでしょうか。

ここで初めて葛藤が生まれます。

つまり、キャラクターのドラマは「ルールがあること」ではなく、「複数のルールが同時には満たせない状況」で強く表れます。

---

## より体系化するなら、「人は何を最優先するか」という観点で整理できます

私は、判断ルールは次の6つの大分類に集約できると考えます。

| 判断の基準  | 典型的な問い          | 例                  |
| ------ | --------------- | ------------------ |
| **善悪** | 「正しいか？」         | 約束を守る、嘘をつかない、弱者を守る |
| **結果** | 「利益になるか？」       | 効率、成功、生存、損得        |
| **関係** | 「誰のためか？」        | 家族、恋人、仲間、恩人を優先する   |
| **自己** | 「自分らしいか？」       | 誇り、自由、美学、信念を守る     |
| **感情** | 「どう感じるか？」       | 好き嫌い、怒り、恐れ、愛情      |
| **探求** | 「何を知り、何を得られるか？」 | 好奇心、成長、真実、挑戦       |

この枠組みなら、キャラクターを「性格」ではなく**判断アルゴリズム**として設計できます。

あなたは人物を「反応の一貫性」で理解するタイプだと話していました。その視点なら、「この人はどの基準を、どんな優先順位で採用するのか」を決める方が、「内面」や「性格」を直接考えるよりもずっと扱いやすいはずです。背景や心理は、その判断アルゴリズムが**なぜ形成されたのか**を説明するものとして後から補えば、一貫性のあるキャラクターになります。
